# Pronunciation data license

`pronunciations.json` in this directory is a **derivative database** built from
the **Quizbowl Pronouncing Dictionary** by **Ophir Lifshitz**:

- Source: https://minkowski.space/quizbowl/pronouncing-dictionary/index.html

It contains a filtered, normalized subset of that dictionary (the
maintainer-verified "OL" entries, reduced to term → pronunciation pairs), and is
regenerated by `python manage.py build_pron_dict`.
