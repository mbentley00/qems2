Upload one or more packet files. A new tournament is created with one packet per file (named after the file). Three formats are supported:

  • .json — the JSON from YetAnotherPacketParser / MODAQ. Recommended: formatting (bold/underline/italics/superscript), powers, and categories (from each question's metadata) come through reliably.
  • .docx / .pdf — best-effort plain-text parsing of the standard ACF layout (numbered tossups + ANSWER: lines, numbered bonuses with [10] parts). Real packets vary, so use the JSON export when you have it.

Questions that can't be parsed are skipped and listed below with a reason, so you can fix them by hand afterward.
