About QEMS3

QEMS3 (the Quizbowl Editing and Management System) is a web application for writing, editing, and packetizing quizbowl question sets. Writers submit tossups and bonuses, editors review and lock them, and set owners manage the distribution and export finished packets.

QEMS3 is an evolution of QEMS2, originally developed by Jerry Vinokurov, Mike Bentley, and others, which itself was an evolution from QEMS primarily developed by Will Butler.

What you can do

  • Submit questions — write ACF-style tossups and bonuses (and VHSL bonuses) with a rich-text editor that produces the standard quizbowl markup.
  • Edit and review — comment on questions, resolve discussions, track an activity feed, and lock questions when they are done.
  • Manage distributions — define per-packet category quotas and watch them fill as questions come in.
  • Packetize — auto-distribute questions into packets, then fine-tune with the drag-and-drop packet grid and document view.
  • Check quality — run the duplicate checker and the style checker before export.
  • Import and export — bring in existing material from TSV, JSON (YAPP/MODAQ), .docx, or PDF, and export packets for distribution.

Roles

Access follows a simple hierarchy: Owner > Editor > Writer. Owners create question sets and assign editors and writers. Editors can edit and lock any question in their set. Writers can edit their own unlocked questions.

QEMS3 is open source. Found a bug or have a suggestion? Report it on GitHub. New here? See the Help page to get started.
