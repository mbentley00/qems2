Help

This page covers the basics of using QEMS3. For what the system is and who it's for, see the About page.

Getting started

  1. From the Home page you'll see the question sets you own, edit, or write for.
  2. Owners can create a question set and set up a distribution (the category breakdown).
  3. Open a set to add questions, manage members, and packetize.

Writing questions

  • Use Add a Tossup / Add a Bonus on a set, or Type Questions to paste several at once.
  • The editor toolbar produces standard quizbowl markup: B = required answer (bold + underline), U = underline (prompt/alternate), I = italics, and subscript / superscript.
  • Mark the power point with (*) in tossups.
  • Answer lines need an underlined required portion. Put pronunciation guides in parentheses after hard-to-say words.

Reviewing and comments

  • Click any span of a question to leave an anchored comment, like a document comment. Reply to and resolve threads, and @mention other members to notify them.
  • The Activity feed shows mentions and recent changes to your questions.
  • Editors lock a question when it's finished; locked questions can no longer be edited by writers.

Packetizing

  • Packetize a set to auto-distribute questions into packets according to the distribution. Overflow goes to an Extras packet.
  • The packet grid lets you drag to swap questions between packets; the document view lets you reorder within a packet and copy out to Discord.
  • Packet-level changes can be undone from the packet grid change log.

Quality checks

  • Duplicate checker — flags repeated answerlines and topic repeats across the set.
  • Style checker — checks each question against a style guide (default: Minkowski) for mechanical and formatting issues.

Importing

Administrators can import existing material from the Home page: Import Set from TSV (a round-tripped QEMS export) or Import Packets, which creates a new set with one packet per uploaded .json (YAPP/MODAQ), .docx, or .pdf file.

Still stuck or found a bug? Open an issue on GitHub.
